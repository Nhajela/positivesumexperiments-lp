The axioms — numbered as additions, because the manifesto is a sum
+1

Business is a model.

A resource-allocation exercise. Everything inside it is a relationship and a transaction.

+2

Only play positive-sum games.

Every transaction is a game. Zero-sum games are not played here. Negative-sum games, never.

+3

Respect time above all else.

Time is limited, so the bets are audacious. There is no point doing work that isn’t.

Dialogue block — the objection, answered plainly

“So this is a startup studio?”

No. It’s one person running serious experiments under one name, with one rule about how every game gets played. Some experiments become companies. All of them have to earn their keep in 30 days.
