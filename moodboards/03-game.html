Dialogue block — the seminar objection

“Isn’t every deal zero-sum if you negotiate hard enough?”

Only if the pie is fixed. The whole bet of this umbrella is that the pie isn’t — that the model can be arranged so the value created outweighs what anyone gives up. When it can’t be arranged, we don’t play.
